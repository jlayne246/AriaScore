# AriaScore Landing Page

A Vite + React landing page using real AriaScore screenshots and the app logo.

## Local development

```bash
cd landing
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Vercel

Set the Vercel project Root Directory to:

```txt
landing
```

Use:

```txt
Build Command: npm run build
Output Directory: dist
```
